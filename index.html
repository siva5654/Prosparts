How Can We Help

Parts Number Search : Manufacturers numbers can take many forms. If the number is long and separated by spaces, dashes ( - ), slashes ( / ) or dots ( . ) or any other non numeric or alphabetic character, it is best to take a core four digit number and enter it in the box on the left. An example would be a number like 45.5565/AB-01. This number could be entered as 5565. PROS part numbers are almost always in the format of three digits separated by a dash and followed by another three digits such as 150-001.

Keyword Search : It is possible to search the PROS database by using keywords which might be found in the description of the part. This process should be the last resort because finding the correct descriptive term that we used to describe the part can be tricky. Start with one word and if too many results are displayed, try entering two words separated by a comma. Example: Enter "Steam" to reveal over 1,000 results but entering "Steam, Valve" narrows the results to less than one hundred
